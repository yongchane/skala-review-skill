# 웹 AI용 프롬프트

ChatGPT·Gemini·Claude 웹에서는 로컬 Skill을 설치하는 대신 목적에 맞는 프롬프트를 복사해 사용합니다. GitHub 링크만 AI에 전달하지 말고, 아래 파일에서 `복사할 프롬프트` 코드 블록 전체를 복사해 새 대화의 첫 메시지로 보내세요.

## 어떤 프롬프트를 사용할까요?

| 원하는 작업 | 사용할 파일 |
|---|---|
| 질문을 주고받으며 복습하고, 이후 플랫폼용 글로 정리 | [질문 복습 프롬프트](skala-review-prompt.md) |
| 질문 과정 없이 메모를 빠르게 요약하거나 플랫폼별 글 구조 작성 | [빠른 정리·작성 프롬프트](skala-quick-draft-prompt.md) |

두 프롬프트 모두 다음 원칙을 따릅니다.

- 비전공자도 이해할 수 있는 쉬운 한국어 사용
- 제한된 강의 자료 전체보다 직접 작성한 요약과 필요한 일부 내용 우선
- 확인되지 않은 수업 내용·출처·분석 결과를 임의로 생성하지 않음
- Velog·Notion·Tistory에 자동 게시하지 않고 복사 가능한 결과만 제공
- 기본 초안은 짧게 제공하고, 사용자가 요청할 때만 자세히 확장

## 수정하고 기여하기

프롬프트는 MIT 라이선스로 공개되어 있어 자신의 학습 방식에 맞게 수정할 수 있습니다. 다른 교육생에게도 도움이 될 개선이라면 Fork 후 프롬프트 파일을 수정하고 Pull Request를 보내주세요.

수정하기 쉬운 항목은 다음과 같습니다.

- 복습 질문 수
- 설명 난이도
- 기본 초안 분량
- 지원할 글쓰기 플랫폼
- 플랫폼별 제목·목차·말투·자료 배치 방법

기여할 때는 실제 강의 원문, 개인정보, 비공개 발언 또는 개인 학습 기록을 예시로 포함하지 마세요. 자세한 절차는 [기여 가이드](../CONTRIBUTING.md)를 확인해 주세요.
