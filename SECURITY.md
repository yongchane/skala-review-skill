# 보안 및 개인정보 제보

## 지원 범위

현재 `main` 브랜치의 최신 버전을 우선 지원합니다. 오래된 설치본에서 문제가 발생하면 최신 버전에서도 재현되는지 먼저 확인해 주세요.

## 비공개로 알려야 하는 문제

- API 키, 토큰 또는 계정 정보 노출
- 사용자의 로컬 파일이 동의 없이 저장되는 문제
- 제한된 강의 자료나 개인정보가 외부로 복사되는 문제
- 임의 파일 접근이나 명령 실행으로 이어질 수 있는 취약점

이런 내용은 공개 Issue에 세부 정보를 작성하지 마세요. GitHub 저장소의 `Security` 메뉴에서 `Report a vulnerability`를 사용해 비공개로 제보해 주세요. 해당 메뉴를 사용할 수 없다면 공개 Issue에는 세부 내용 없이 비공개 연락 방법을 요청해 주세요.

제보에는 가능한 범위에서 영향받는 버전, 재현 절차, 예상 결과와 실제 결과를 포함해 주세요. 실제 개인정보나 비밀 값은 예시에서도 제거해야 합니다.
