# 복습 노트 표준 형식

노트를 저장할 때 다음 frontmatter 형식을 사용한다. 필드명과 유형 값은 스크립트가 읽는 식별자이므로 변경하지 않는다.

```yaml
---
title: ""
date: YYYY-MM-DD
profile: concept-review | lecture-recap | practice-log | comparison-note | troubleshooting-note
keywords: []
status: reviewed
source: user-provided-summary
skala_module: ""
---
```

본문에는 다음 내용을 포함한다.

- 제목과 사용자가 이해한 핵심 정리
- 예시, 복습 문답, 오개념 수정 또는 실습 결과 중 최소 하나
- 복습 중 다룬 경우 SKALA 과정 또는 프로젝트와의 짧은 연결
- 다음 복습 질문이나 후속 행동

선택한 유형에 필요한 목차만 사용한다. 나중에 검색할 수 있도록 `keywords`에는 구체적인 단어를 넣는다. 원본 PDF, 긴 강의 원문, 개인정보 또는 배포가 제한된 자료를 포함하지 않는다.
