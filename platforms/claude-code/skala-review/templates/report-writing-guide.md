# 실습 보고서 작성 가이드

## 과제 요구사항

- 목표:
- 필수 결과물:
- 평가 기준:

## 추천 목차

1. 분석 또는 실습 목적
2. 진행 방법
3. 결과
4. 결과 해석과 인사이트
5. 한계와 다음 단계

## 항목별 작성 방법

각 항목에 무엇을 써야 하는지와 그 근거를 짧게 설명한다. 사용자가 직접 산출한 수치·표·그래프·코드가 들어갈 위치를 표시한다.

## 짧은 문장 초안

사용자가 제공한 결과만 문장으로 정리한다. 제공되지 않은 값은 `[실행 결과 입력]`, 확인되지 않은 근거는 `[근거 확인 필요]`로 남긴다.
