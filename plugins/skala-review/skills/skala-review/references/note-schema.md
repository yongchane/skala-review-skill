# Canonical review note schema

Use this frontmatter when saving a note.

```yaml
---
title: ""
date: YYYY-MM-DD
profile: concept-review | lecture-recap | practice-log | comparison-note | troubleshooting-note
keywords: []
status: reviewed
source: user-provided-summary
---
```

The body must contain a title, the learner's key takeaway, and either an example, a question-and-answer record, or a practice result. Add only sections justified by the selected profile.
